# 用户体验规格 (UX Specification)

## 1. 设计原则

### 1.1 核心原则
- **苏格拉底式体验**: 界面和交互设计应体现引导而非灌输的理念
- **简洁优先**: 减少认知负担，让用户专注于思考
- **即时反馈**: 所有操作都有明确的状态反馈
- **包容性**: 支持不同技能水平的用户

### 1.2 视觉风格
- **主题**: 温暖、学术、专注
- **主色**: #4A6FA5 (沉稳蓝)
- **强调色**: #E8B339 (思考黄)
- **背景色**: #F8F9FA (浅色) / #1A1A2E (深色)
- **成功色**: #28A745
- **错误色**: #DC3545
- **字体**: 
  - 中文: "Noto Sans SC", system-ui
  - 英文: "Inter", system-ui

## 3. 组件设计

### 3.1 登录/注册页面

**布局**:
- 居中卡片 (最大宽度 400px)
- Logo + 标题
- 表单区域
- 第三方登录按钮
- 底部切换链接

**表单字段**:
- 邮箱输入框 (带验证提示)
- 密码输入框 (显示/隐藏切换)
- 名字输入框 (注册时)
- 角色选择下拉框 (注册时)

**交互状态**:
- 输入验证 (实时)
- 提交按钮加载态
- 错误信息提示 (红色)
- 成功跳转动画

### 3.2 仪表盘 (Dashboard)

**布局**:
- 顶部: 欢迎语 + 学习统计
- 中间: 快速开始卡片
- 底部: 最近会话列表

**组件**:
1. **统计卡片** (StatCard)
   - 图标 + 数字 + 标签
   - 悬停放大效果

2. **快速开始** (QuickStart)
   - 学科选择按钮
   - 新建会话入口

3. **最近会话列表** (SessionList)
   - 卡片式列表
   - 显示: 标题、学科、最后活动时间
   - 悬停显示操作菜单

### 3.3 对话页面 (核心页面)

**布局** (桌面端):
```
┌─────────────────────────────────────────────┐
│ Header: 会话标题 | 学科标签 | 设置按钮       │
├─────────────────────────────────────────────┤
│                                             │
│              消息列表区域                    │
│   ┌─────────────────────────────────┐      │
│   │ 🤖 导师消息 (左对齐)            │      │
│   │ "你觉得这个问题应该从..."       │      │
│   │ [提示词1] [提示词2]             │      │
│   └─────────────────────────────────┘      │
│                                             │
│        ┌─────────────────────────┐          │
│        │ 用户消息 (右对齐)       │          │
│        │ "我认为是..."           │          │
│        └─────────────────────────┘          │
│                                             │
├─────────────────────────────────────────────┤
│ Input: [输入框] [发送按钮] [停止按钮]       │
└─────────────────────────────────────────────┘
```

**消息气泡样式**:
- **导师消息**:
  - 背景: #E8EEF4 (浅蓝灰)
  - 左侧箭头
  - 底部显示提示词按钮
  
- **用户消息**:
  - 背景: #4A6FA5 (主色)
  - 文字: 白色
  - 右侧箭头

**输入区域**:
- 多行文本输入 (支持Shift+换行)
- 发送按钮 (Enter快捷键)
- 停止按钮 (流式响应时显示)
- 字数统计 (当前/最大)

**边缘情况**:
- 空输入: 禁用发送按钮
- 输入中: 显示"正在思考..."
- 响应中: 显示加载动画 + 停止按钮
- 错误: 显示错误提示 + 重试按钮

### 3.4 历史会话页面

**布局**:
- 顶部筛选栏 (状态、学科、日期)
- 会话卡片网格 (每行3个桌面端)

**卡片内容**:
- 学科图标
- 会话标题
- 预览文字 (首条消息)
- 时间戳
- 消息数
- 操作: 继续 / 归档 / 删除

### 3.5 学习进度页面

**布局**:
- 顶部: 总体统计
- 中间: 能力雷达图
- 底部: 概念掌握列表

**组件**:
1. **能力雷达图** (RadarChart)
   - 各学科能力值可视化
   
2. **掌握度进度条** (MasteryBar)
   - 四色渐变 (beginner→mastery)
   
3. **学习时间线** (Timeline)
   - 按日期展示学习活动

### 3.6 设置页面

**分组**:
1. 个人信息 (头像、名字、邮箱)
2. 学习偏好 (学科排序、对话风格)
3. 界面设置 (主题、语言、字体大小)
4. 账户安全 (修改密码、退出登录)

## 4. 交互动效

### 4.1 过渡效果
- 页面切换: fade + slide (200ms)
- 卡片出现: fade-in (150ms, stagger 50ms)
- 按钮悬停: scale(1.02) + shadow

### 4.2 消息动画
- 新消息: slide-up + fade-in (300ms)
- 流式输出: 逐字显示 (50ms/字)

### 4.3 加载状态
- 骨架屏 (Skeleton): 用于列表加载
- 脉冲动画: 用于按钮加载
- 旋转图标: 用于全局加载

---

## 5. 错误处理

### 5.1 网络错误
- Toast提示: "网络不稳定，请检查连接"
- 自动重试按钮
- 离线模式提示

### 5.2 API错误
- 401: 跳转登录页
- 403: 显示权限不足提示
- 500: "服务器错误，请稍后重试"

### 5.3 AI响应错误
- "导师暂时走神了，请再试一次"
- 提供手动重试按钮

## 6. 辅助功能

### 6.1 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| Enter | 发送消息 |
| Shift+Enter | 换行 |
| Ctrl+K | 快速跳转 |
| Escape | 关闭弹窗 |

### 6.2 无障碍
- ARIA标签完整
- 焦点管理
- 颜色对比度 > 4.5:1
- 支持屏幕阅读器
